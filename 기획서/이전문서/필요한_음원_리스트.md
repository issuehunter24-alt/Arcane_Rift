## 🎵 음원 요구사항

### SFX (효과음)
- **형식**: MP3, OGG, WAV 모두 가능 (MP3 권장)
- **길이**: 0.2초 ~ 2초 (짧고 간결하게)
- **크기**: 개당 50KB 이하 권장
- **비트레이트**: 128kbps ~ 192kbps
- **샘플레이트**: 44.1kHz

### BGM (배경음악)
- **형식**: MP3 권장 (용량 고려)
- **길이**: 
  - menu.mp3: 1~2분 (루프 가능하게)
  - battle.mp3: 2~3분 (루프 가능하게)
  - shop.mp3: 1~2분 (루프 가능하게)
  - victory_jingle.mp3: 5~10초 (짧은 축하 곡)
- **크기**: 곡당 2MB 이하 권장
- **비트레이트**: 128kbps (용량 절약)
- **루프 설정**: 자연스럽게 반복 가능하도록 편집

## 🎼 추천 무료 음원 사이트

### 1. Pixabay (가장 추천)
- **URL**: https://pixabay.com/music/
- **라이선스**: 완전 무료, 상업용 가능
- **특징**: 고품질, 다운로드 쉬움, 저작권 걱정 없음
- **검색 키워드**:
  - SFX: "game sound", "button", "card", "magic", "hit", "heal"
  - BGM: "game music", "fantasy", "battle", "menu", "victory"

### 2. Mixkit
- **URL**: https://mixkit.co/free-sound-effects/game/
- **라이선스**: 완전 무료
- **특징**: 게임 효과음 특화, 카테고리 분류 잘 됨

### 4. ZapSplat
- **URL**: https://www.zapsplat.com/
- **라이선스**: 무료 (계정 필요)

## 🔧 음원 적용 방법

1. **다운로드한 음원을 폴더에 배치**
   ```
   web/public/sounds/sfx/  ← 효과음 17개
   web/public/sounds/bgm/  ← 배경음악 4개
   ```

2. **파일명 정확히 맞추기**
   - 위 리스트의 파일명과 정확히 일치해야 함
   - 예: `damage.mp3`, `battle.mp3` (대소문자 구분)

3. **브라우저에서 자동 로드**
   - 게임 실행 시 `audio.ts`에서 자동으로 등록
   - 로딩 실패 시 콘솔에 경고 표시됨

4. **테스트**
   - 게임 실행 후 F12 → Console에서 `[Audio]` 로그 확인
   - 우하단 🔊 버튼으로 볼륨 조절 테스트
