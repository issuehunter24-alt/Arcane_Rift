## UX/UI 가이드

### 1. 핵심 화면
- 메인 화면: 상단 유틸(프로필/재화/설정), 중앙 주요 모드(캠페인/일일/드래프트[옵션]/PvP[옵션]), 하단 빠른 액션(덱 편집/상점/아카이브).
- 전투: 선언→락인→공개 단계가 명확하도록 시각/음향 큐 제공. 라운드 타이머, 에너지, 이니셔티브 인디케이터.
- 덱 편집: 타입/등급/코스트 필터, 코스트·타입 비율 그래프, 중복 카운트 표시, 미리보기.
- 보상/상점: 드랍/가격 정보, 키워드 툴팁, 일일 할인 배지, 미리보기 확대.

### 2. 접근성/모바일
- 텍스트 크기 옵션, 고대비, 색각 보정 프리셋.
- 터치 목표 48dp 이상, 드래그/탭/롱탭 제스처 표준화.

### 3. 가독성
- 카드: 이름 1줄(최대 18자), 효과 2줄 요약 + 상세 툴팁, 키워드 하이라이트.
- HUD: 에너지, 라운드, 이니셔티브는 항상 가시 영역 우상단 고정.

### 4. 화면 플로우(요약)
- 메인 → (캠페인 | 일일 | 덱 편집 | 상점 | 아카이브 | 설정)
- 캠페인 → 스테이지 선택 → 전투 → 보상 → 메인
- 덱 편집 → 카드 라이브러리 → 덱 저장 → 메인
- 상점 → 구매 → 확인 토스트 → 메인

### 5. 메뉴 구성(요약)
- 메인 상단: 프로필(닉네임/레벨), 재화(골드/조각/티켓), 알림, 설정.
- 메인 중앙: 캠페인, 일일, [드래프트/PvP 옵션], 이벤트 배너.
- 메인 하단: 덱 편집, 상점, 아카이브(컬렉션/도감), 뉴스.
