## 디자인 스타일 가이드 (중세 판타지 테마)

### 1) 아트 디렉션 키워드
- 중세 판타지, 기사단, 성당 스테인드글라스, 고서(양피지), 금속/가죽, 문장학(Heraldry)
- 톤: 장중함, 고급스러움, 절제된 화려함(과도한 장식 금지)

### 2) 컬러 팔레트
- 기본: 딥 네이비(#0F1A2C), 차콜 블랙(#1A1A1A), 고서 베이지(#D9C9A3)
- 포인트(등급별):
  - Normal: 철회색(#9AA4AE)
  - Rare: 사파이어 블루(#2E6DB4)
  - Epic: 자수정 퍼플(#6B4FA3)
  - Legendary: 금박 골드(#CFAE3C)
- 경고/강조: 승리 그린(#8ACB88), 패배 크림슨(#B23A48)

### 3) 타이포그래피
- 제목/카드명: 세리프(중세풍) 우선 — Noto Serif KR, Cormorant Garamond(영문)
- 본문/효과: 고가독 산세리프 — Noto Sans KR/Inter
- 대문자/스몰캡은 제한적으로 사용, 자간은 -2% 내외(가독 유지)

### 4) 재질/텍스처
- 배경: 미세한 양피지/캔버스 질감(노이즈 2~4%)
- 프레임: 브러시드 메탈(철/은/금), 가장자리 미세 스크래치
- 버튼: 가죽 바인딩 느낌의 엠보싱, 금속 리벳 하이라이트 최소화

### 5) 아이코노그래피
- 문장학 기반 단순 형상(방패, 검, 성배, 번개, 서책 등)
- 선 굵기 일관(2px @1080p 기준, 스케일 비례), 내부 음영 최소
- 타입 아이콘 예: Attack(검), Heal(성배/십자), Defense(방패), Special(서책/문장)

### 6) 카드 프레임/레어리티 장식
- 공통 규격: 768×1024(내부 컨텐츠 세이프존 680×940)
- 레어 테두리: Normal(철), Rare(강철+블루 인레이), Epic(자수정 인레이), Legendary(금박+엔그레이빙)
- 보석/인레이는 한 면 기준 1~2개로 절제, 광택 하이라이트는 15% 이하

### 7) UI 컴포넌트
- 버튼: 가죽 질감 + 금속 프레임, 호버 시 4% 밝기↑, 프레스 시 내부 그림자 2px
- 패널/모달: 양피지 바탕 + 금속 코너 가드, 투명도 92~96%
- 툴팁: 암청 배경 + 금박 라인 1px, 키워드 컬러 하이라이트(속성 색 매핑)

### 8) 레이아웃/그리드
- 기본 12컬럼, 거터 24px(1080p 기준). 카드 그리드는 3열(PC), 2열(모바일)
- 상단 유틸 바는 64px 높이, 우측 정렬(프로필/재화/설정)
- 중요 정보(에너지/이니셔티브)는 우상단 고정, 대비비 4.5:1 이상

### 9) 애니메이션/전이
- 전투: 선언→락인→공개 전환 시 페이드/슬라이드 150~300ms
- 히트 피드백: 120~180ms 흔들림/플래시, 과도한 블룸 금지
- UI: 호버 100ms 이내, 모달 오픈 200ms 페이드

### 10) 사운드/연출 톤 정합(참고)
- 금속성, 현악/합창 계열 음색. UI 클릭은 목재/가죽 터치감
- 이펙트는 오디오/이펙트 가이드의 매핑 표와 컬러 매칭 유지

### 11) 접근성
- 텍스트 최소 크기: 본문 16px(1080p), 모바일 14px 이상
- 대비: 주요 텍스트/아이콘 4.5:1, 버튼/상태 표시 3:1 이상
- 색약 대응: 속성 색에 보조 패턴(문양) 부여 가능

### 12) 사용/비사용 예(Do/Don't)
- Do: 절제된 금속 하이라이트, 문장학 패턴, 양피지 질감의 은은한 노이즈
- Don't: 과도한 글로스/네온, 사이버/현대 UI 요소(글래스모피즘 등), 과한 입체 드롭쉐도우

### 13) 산출물 규격/내보내기
- 아이콘: SVG 우선, PNG @1x/1.5x/2x 세트(WebP 동시 산출)
- 프레임/배경: PNG/WebP, 투명 배경 지원, 8px 세이프 마진
- 폰트: 웹 임베드 라이선스 확인, 폰트 폴백 체인 정의

### 14) 검수 체크리스트
- 컬러 매핑 일치(등급/속성), 대비 기준 충족, 해상도별 스케일 정상
- 타이포 위계 준수(제목/본문/보조), 텍스처 과도 사용 금지
- 전투 HUD 가독성, 카드명 오버플로우 처리(말줄임/축약)
