# 화면 배경 리소스 설계서

## 📋 게임명
**"빈틈없는 악역이 회귀했다면, 이번엔 가문 인정받을 때까지 그만둘 수 없어"**

---

## 🎨 디자인 컨셉

### 전체 테마
- **판타지 귀족 시대**: 중세 판타지 배경, 벨몬트/드레이크 두 귀족 가문 중심
- **마법 카드 배틀**: 카드로 마법을 소환하고 전투하는 세계관
- **회귀물 설정**: 세라피나의 전생 기억과 성장 스토리
- **고급스러운 분위기**: 우아하고 화려한 귀족 미학

### 컬러 톤
- **주요 컬러**: 보라, 은색, 금색
- **배경 컬러**: 어두운 틴트(#0F1A2C, #1a2f4f)
- **강조 컬러**: #4a9eff (파란색), #9c27b0 (보라색)
- **분위기**: 우아하고 신비로운, 약간 애니메이션 스타일

---

## 🖼️ 화면별 배경 리소스 설계

### 1. **메인 메뉴 화면** (Menu Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 벨몬트 가문 저택의 화려한 현관 홀

**요소**:
- 귀족 저택의 화려한 계단과 현관
- 천장에서 내려오는 샹들리에
- 양쪽에 벨몬트 가문의 문장/깃발
- 배경에 보라빛 마법 구슬이나 카드가 떠다님
- 약간 어두운 분위기 (밤 또는 황혼)
- 중앙에 제목 표시 영역 (UI 오버레이)

**Variation**: 없음 (고정)

**우선순위**: ⭐⭐⭐ (최상급)

---

### 2. **캠페인 선택 화면** (Campaign Select Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 스테이지가 표시되는 지도/계획판

**요소**:
- 마법으로 떠있는 홀로그램 지도
- 스테이지마다 보라/금색 마크
- 벨몬트 가문의 서재 배경
- 오른편에 항해 일지나 계획서
- 지도 위에 카드들이 배치됨 (클리어 시 하이라이트)

**Variation**: 없음 (고정)

**우선순위**: ⭐⭐⭐ (최상급)

---

### 3. **덱 편집 화면** (Deck Editor Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 마법 연구실/서재

**요소**:
- 책상 위에 카드들이 흩어져 있음
- 뒤편에 마법서와 두루마리
- 보라빛 마법 오라가 배경에 감돔
- 좌우에 서가와 기물들
- 중심에 카드 정렬 영역 (UI)

**Variation**: 없음 (고정)

**우선순위**: ⭐⭐ (중간)

---

### 4. **전투 화면** (Battle Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 스테이지별 배경 (총 50개 스테이지 × 3 Variations = 150장)

**현재 스테이지 배경**:
1. `stage_01_training_1/2/3.png` (훈련장)
2. `stage_02_fire_1/2/3.png` (화염 연습장)
3. `stage_03_ice_1/2/3.png` (얼음 동굴)
4. `stage_04_lightning_1/2/3.png` (뇌전의 탑)
5. `stage_05_wind_1/2/3.png` (바람의 신전)
6. `stage_06_fire_ice_1/2/3.png` (화염과 빙설의 폭풍)
7. `stage_07_storm_1/2/3.png` (폭풍의 전장)
8. `stage_08_elite_1/2/3.png` (벨몬트 대전투장)
9. `stage_09_corridor_1/2/3.png` (어둠의 관문)
10. `stage_10_drake_estate_1/2/3.png` (드레이크 저택)
... (총 50 스테이지)

**각 배경 요구사항**:
- 좌측 하단: 플레이어 HP/에너지 영역
- 우측 하단: 적 HP/에너지 영역
- 중심: 전투 씬
- 배경에 마법 이펙트
- 스테이지 테마에 맞는 분위기

**Variation**: 3종 (랜덤, 순차, 난이도별)

**우선순위**: ⭐⭐⭐ (최상급, 각 스테이지마다 중요)

---

### 5. **컷신 화면** (Cutscene Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 스테이지별 전투 전/후 스토리 배경

**요소**:
- 배경과 캐릭터 일러스트 결합
- 대화창이 중앙 하단에 표시
- 캐릭터 초상화 우측/좌측 상단
- 배경은 전투 배경과 유사하되 더 정적

**Variation**: 대부분 전투 배경과 공유

**우선순위**: ⭐⭐ (중간)

---

### 6. **승리 화면** (Victory Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 화려한 승리 연출

**요소**:
- 금빛/보라빛 마법 폭죽
- 주인공이 카드로 승리하는 연출
- 배경이 어둡고 중심만 밝음
- UI 오버레이 (승리 제목, 보상 버튼)

**Variation**: 3종 (random variation)

**우선순위**: ⭐⭐ (중간)

---

### 7. **패배 화면** (Defeat Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 어두운 패배 연출

**요소**:
- 어둡고 침울한 배경
- 부서진 카드나 마법 잔재
- 약간 붉은 틴트
- UI 오버레이 (패배 제목, 재도전 버튼)

**Variation**: 3종 (random variation)

**우선순위**: ⭐⭐ (중간)

---

### 8. **상점 화면** (Shop Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 마법 상점/카드 딜러

**요소**:
- 화려한 마법 상점 인테리어
- 카드가 유리 진열장에 정렬
- 상인 NPC 일러스트 (선택)
- 물건들과 캐릭터들이 뒤편에 배치
- 우측에 재화 표시 영역

**Variation**: 없음 (고정)

**우선순위**: ⭐ (낮음)

---

### 9. **보상 화면** (Reward Screen)
**사이즈**: 1920 x 1080 (Full HD, 또는 1024 x 768)

**컨셉**: 보상이 떠있는 마법 오라

**요소**:
- 어두운 배경에 보라빛/금빛 오라
- 보상 카드가 중심에 떠있음
- UI 오버레이 (보상 아이템 표시)
- 텍스처는 간단하게

**Variation**: 없음 (고정)

**우선순위**: ⭐ (낮음)

---

## 📐 리소스 사이즈 최적화 전략

### 최대 제약: 1024 x 1024

### 권장 해상도:
1. **Full HD**: 1920 x 1080 (16:9) → 권장하지 않음 (제약 초과)
2. **1280 x 720** (720p, 16:9) → 권장하지 않음 (제약 초과)
3. **1024 x 768** (4:3) → 권장
4. **1024 x 576** (16:9) → 권장
5. **800 x 450** (16:9) → 권장
6. **720 x 480** (3:2) → 최소 품질

### 각 화면별 최적 사이즈:

| 화면 | 권장 사이즈 | 우선순위 | 파일 수 |
|------|------------|---------|---------|
| 메인 메뉴 | 1024 x 576 (또는 1024 x 768) | ⭐⭐⭐ | 1 |
| 캠페인 선택 | 1024 x 576 (또는 1024 x 768) | ⭐⭐⭐ | 1 |
| 덱 편집 | 1024 x 576 | ⭐⭐ | 1 |
| 전투 (스테이지별) | 1024 x 576 | ⭐⭐⭐ | 150 (50 × 3) |
| 컷신 | 1024 x 576 | ⭐⭐ | 공유 또는 최소 |
| 승리 | 1024 x 576 | ⭐⭐ | 3 |
| 패배 | 1024 x 576 | ⭐⭐ | 3 |
| 상점 | 1024 x 576 | ⭐ | 1 |
| 보상 | 1024 x 576 | ⭐ | 1 |

## 📝 구현 가이드

### 제작 방식 옵션:
1. **AI 생성**: Midjourney, Stable Diffusion 등으로 창작
2. **마켓플레이스**: Unity Asset Store, Sketchfab 등에서 구매
3. **커스텀 아트**: 외주 또는 직접 제작

### 스타일 가이드:
- **여성향 판타지**: 우아하고 화려한 느낌
- **일본 애니메이션 스타일**: 부드러운 라인, 페이스탈 컬러
- **3D 또는 2D**: 둘 다 가능, 일관성만 유지
- **실사 vs 스타일화**: 스타일화 권장 (게임 톤에 맞게)

### 파일 형식:
- **최종 배포**: WebP (최적화)
- **원본 작업**: PNG (투명도 가능)
- **압축**: 최대 200KB 이하 권장 (WebP 기준)
