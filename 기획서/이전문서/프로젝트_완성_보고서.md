## 🎨 기술 스택 상세

### **Frontend**
- **TypeScript 5.6.3** - 타입 안전성
- **Vite 5.4.8** - 빌드 도구
- **PixiJS 8.2.1** - 2D WebGL 렌더링
- **Zustand 4.5.2** - 상태 관리

### **Testing**
- **Vitest 1.5.0** - 단위 테스트
- **@vitest/ui** - 테스트 UI

### **Performance**
- **Object Pooling** - 카드 컨테이너 재사용
- **Asset Preloading** - WebP + PNG 폴백
- **Rendering Batching** - requestAnimationFrame
- **Resolution Scaling** - 저사양 디바이스 대응
