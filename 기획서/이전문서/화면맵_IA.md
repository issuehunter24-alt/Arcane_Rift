## 화면 맵 & 정보 구조(IA)

### 1. 최상위 구조
- 메인
  - 캠페인
    - 스테이지 선택 → 전투 → 보상
  - 일일 던전
    - 속성 선택 → 전투 → 보상
  - 드래프트/아레나(옵션)
    - 덱 구성 → 매치 → 보상
  - PvP(옵션)
    - 랭크/빠른 대전 → 매치 → 보상
  - 덱 편집
    - 라이브러리 → 덱 저장
  - 상점
    - 일반/특가/코스메틱
  - 아카이브(컬렉션/도감)
  - 설정

### 2. 전투 하위 구조
- 상단: 라운드/이니셔티브/에너지, 타이머
- 중앙: 보드/카드/상태이상 아이콘
- 하단: 핸드/선언 큐/락인 버튼/공개 애니메이션

### 3. 데이터 흐름
- 카드 DB(JSON) → 덱 편집/전투 로딩 → 전투 로그/텔레메트리 → 대시보드/밸런스 피드백 → 데이터 업데이트
