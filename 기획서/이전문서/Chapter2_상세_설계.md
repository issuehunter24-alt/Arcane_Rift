## Stage 11: Lucian Rosegarden (Round 2)
- **테마**: 실력 향상 확인, 중급 마법
- **권장 파워**: 220
- **보상**: 골드 400, 파편 4 / 골드 200, 파편 2
- **배경**: 훈련장 (나중 시간대, 저녁)
- **배경 이미지**: `stage_11_training_evening_1.png` (기존 훈련장 재사용 가능)

### 스토리 핵심
세라피나의 실력 향상을 확인하는 재검증 배틀. 첫 배틀보다 훨씬 강해진 Lucian과 싸워 중급 마법사 수준을 증명한다.

### 전투 전 대화
```
Lucian: "아가씨, 정말 빠르게 성장하고 계시는군요. 이제 중급 마법사 수준을 테스트해보겠습니다."
Seraphina: "네, Lucian. 제 성장을 확인해보세요."
Lucian: "이번엔 더 강한 마법을 사용하겠습니다. 준비 되셨나요?"
```

### 전투 후 승리
```
Lucian: "대단합니다! 이 정도면 확실히 중급 마법사 급이네요."
Seraphina: "고마워요. 계속 노력하겠어요."
Lucian: "다음 시험도 기대됩니다, 아가씨."
```

### 전투 후 패배
```
Lucian: "아직 조금 부족하시네요. 더 수련이 필요해 보입니다."
Seraphina: "알겠어요. 다시 도전하겠습니다."
```
