## 튜토리얼 전투 시나리오(2~3전)

### 전투 1: 기본 조작/선언-공개 흐름
- 목표: 카드 선언→락인→동시 공개, 타겟 확정 이해.
- 가이드 단계
  1) 카드 선택 안내(하이라이트): "카드를 1장 선택해 선언하세요."
  2) 락인 버튼 활성화 → 클릭 → "양측 선언이 잠겼습니다."
  3) 동시 공개 애니메이션 → 피해 수치 표시 → 결과 로그 표시
- 적 구성: Neutral 공격 카드만 사용(코스트 1~2)
- 승리 조건: 적 HP 0

### 전투 2: 키워드 체험(Burn/Freeze)
- 목표: 화상 도트/빙결 행동감소의 틱/타이밍 이해.
- 가이드 단계
  1) Fire 카드 사용 유도: "파이어볼로 화상을 적용해 보세요."
  2) 턴 시작 틱 데미지 안내: "턴 시작에 화상 피해가 적용됩니다."
  3) Ice 카드 사용 유도: "얼음 가시로 적 행동 -1을 적용해 보세요."
- 적 구성: 체력/방어 높음, 버프 없음
- 승리 조건: 적 HP 0

### 전투 3: 대응/무효/우선순위(선택)
- 목표: Nullify 반응 창, 우선순위(속도>코스트>이니셔티브)의 동작 이해.
- 가이드 단계
  1) 적이 강력한 Attack 선언(코스트 3). 공개 직전 Nullify 사용 안내.
  2) 스택 상단에 Nullify 해결 → 원카드 무효 → 비용 환불 여부 로그 안내.
  3) 우선순위 툴팁: "Special이 Attack보다 먼저 해결됩니다."
- 승리 조건: 적 HP 0(난이도 완화)

UI/UX 노트
- 모든 단계 텍스트는 현지화 키 사용. 하이라이트/펄스 애니메이션으로 유도.
- 스킵 옵션 제공(다음에 다시 보기 가능).
