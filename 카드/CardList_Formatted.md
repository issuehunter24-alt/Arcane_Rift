# 카드 리스트 (Card List) - 보기 편한 버전

## 📋 카드 정보 구조
- **카드번호**: C001 ~ C100
- **카드명**: 카드의 이름
- **타입**: 공격, 회복, 특수, 방어
- **효과**: 카드의 효과 설명
- **코스트**: 사용에 필요한 코스트 (1~3)
- **설명**: 카드의 상세 설명
- **등급**: Normal, Rare, Epic, Legendary
